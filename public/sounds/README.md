# /public/sounds

Place your audio files here.

## Required files

| File | Description | Recommended |
|------|-------------|-------------|
| `ambient.mp3` | Soft looping fantasy piano | ~1–3 min loop, 64kbps |
| `ambient.ogg` | OGG fallback for ambient | Same as above |
| `page-turn.mp3` | Single page-flip sound | ~0.5–1s |
| `magic.mp3` | Magical shimmer / sparkle SFX | ~1–2s |

## Free sources

- https://freesound.org (search "page turn", "ambient piano", "magic shimmer")
- https://pixabay.com/music/ (royalty free)
- https://soundsnap.com

## Notes

- Ambient music should be low volume, non-distracting
- All sounds are triggered only when the user enables audio via the 🎵 toggle
- No autoplay — respects user preference
